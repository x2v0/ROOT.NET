This is ROOT .NET interop interface
provided by two DLLs: ROOT.NET.dll, ROOTdotNET.dll
This allows to embed ROOT canvas into .NET application and execute ROOT macros from .NET.

The API and usage are described in ROOT.NET/ReadMe.txt

It contains several examples of usage hsimple.NET. 
See */ReadMe.txt and the source code for details.

To run it 
- %ROOTSYS% enviroment variable must pont to ROOT installation directory
- %ROOTSYS%\bin must be added to %PATH%
